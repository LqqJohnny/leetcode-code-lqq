### 关键词

- 稳定性 

表示数组中相同的数字， 排序后还是按照原先的先后顺序排列。例如： 1,2,(3),[3],4 ，这里面的 (3) 是在 [3] 之前的 如果排序后变成了 1,2,[3],(3),4  那顺序就错误了， 稳定性算法可以应用于一些场景，比如， 在按价格排序的列表了，要新增一个按照销量排序， 销量相同的两个产品，就按照原先的价格排序。
